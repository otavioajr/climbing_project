INSTRUÇÕES PARA ADICIONAR A FONTE SAILORS

Para que a fonte Sailors funcione corretamente, você precisa adicionar os seguintes arquivos nesta pasta:

1. sailors.woff2
2. sailors.woff
3. sailors.ttf

Você pode baixar esses arquivos da fonte Sailors de um serviço de fontes ou usar o arquivo original que você possui.

Após adicionar os arquivos, a fonte será carregada automaticamente no projeto.
